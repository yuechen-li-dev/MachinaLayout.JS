# Music Player Sample

This sample demonstrates a Claude-generated React music player UI wired through MachinaLayout views using the local root `machinalayout` package.

## Local dependency

This sample depends on the repository root package via:

- `"machinalayout": "file:../.."`

## Run

```bash
# from repo root
npm run build

cd samples/music-player
npm install
npm run dev
```

## Build check

```bash
npm run build
```
